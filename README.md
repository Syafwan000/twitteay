## Installation

1. Clone the project

    ```bash
      git clone https://github.com/Syafwan000/twitteay
    ```

2. Go to the project directory

    ```bash
      cd twitteay
    ```

3. Install dependencies

    ```bash
      npm install
    ```

4. Start the server

    ```bash
      npm run dev
    ```
